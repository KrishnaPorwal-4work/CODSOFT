# 🌸 Iris Flower Classification App

## 📌 Overview

This project is a Machine Learning-based web application that classifies iris flowers into different species based on their physical measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model is trained on the Iris dataset and deployed using Streamlit for real-time, interactive predictions.

---

## 🚀 Features

* 📊 Performed Exploratory Data Analysis (EDA) using data visualization
* 🤖 Built and trained a classification model using Logistic Regression
* 📈 Evaluated model performance using:

  * Confusion Matrix
  * Precision, Recall, and F1-score
* 🌐 Developed an interactive web application using Streamlit

---

## 🧠 Dataset

* Iris Dataset containing **150 samples** across **3 species**:

  * Iris-setosa
  * Iris-versicolor
  * Iris-virginica

### 🔢 Features Used

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## ▶️ How to Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/iris-flower-classification.git
```

### 2️⃣ Navigate to the project folder

```
cd iris-flower-classification
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
streamlit run app.py
```

## 📊 Model Performance

* Achieved **100% accuracy** on the test dataset
* Zero misclassification observed in the confusion matrix
* Perfect **precision, recall, and F1-score** across all classes
